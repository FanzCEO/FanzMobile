## Troubleshooting

### Messages not appearing
1. Check your internet connection
2. Pull down to refresh
3. Verify sync is enabled in Settings
4. Check if the message source is connected

### AI analysis not showing
- AI processing may take a few seconds
- Check if AI features are enabled in Settings
- Some messages may not have extractable data

### App running slowly
1. Close other apps
2. Clear cache: Settings â†’ Storage â†’ Clear Cache
3. Restart the app

### Can't send messages
- Verify you have the correct permissions
- Check your message quota
- Ensure the contact isn't blocked

### Sync conflicts
If the same message appears twice:
1. Pull down to refresh
2. If persists, go to Settings â†’ Sync â†’ Resolve Conflicts

## FAQ

**Q: Is my data secure?**
A: Yes, all data is encrypted in transit and at rest.

**Q: Can I use WickedCRM offline?**
A: You can view cached messages offline. Sync requires internet.

**Q: How does AI processing work?**
A: Messages are analyzed to extract useful information like dates, tasks, and contacts.

**Q: Will this read all my messages?**
A: Only messages you choose to sync are processed.

**Q: Can I delete individual messages?**
A: Yes, swipe left on any message and tap Delete.
