# Project Summary
The CRM Escort AI project is a web-based dashboard designed to assist escorts in managing their communications, schedules, and client interactions through AI integration. It connects to SMS and calendar services, streamlining the process of responding to messages and scheduling meetings automatically. This application aims to enhance productivity and improve client relationships for escorts by utilizing advanced AI capabilities.

# Project Module Description
The CRM Escort AI project consists of several functional modules:
- **Dashboard**: Displays an overview of messages, contacts, and upcoming events.
- **Messages**: Manages multi-channel communication with AI enhancements for message processing.
- **Contacts**: Provides a smart interface for managing client contacts with tagging and scoring features.
- **Calendar**: Integrates event scheduling and synchronization with external calendars.
- **Workflows**: Automates tasks based on triggers for improved efficiency.
- **Integrations**: Connects with popular services like Twilio, Google Calendar, and Outlook.
- **Settings**: Allows users to manage their profile, API keys, and notification preferences.

- **README.md**: Documentation for project setup and usage.

- **src/App.tsx**: Main application logic and routing.
- **src/components/**: Contains reusable UI components and layout elements.
- **src/pages/**: Individual page components for routing.
- **src/lib/**: API clients and utility functions for data fetching and processing.
- **src/types/**: TypeScript interfaces for type safety.

# Technology Stack
- **Framework**: React 18 + TypeScript
- **UI Library**: shadcn/ui + Tailwind CSS
- **State Management**: Zustand + React Query
- **Routing**: React Router v6
- **HTTP Client**: Axios
- **Build Tool**: Vite

# Usage
### Prerequisites
- Node.js 18+ and pnpm
- Backend API running (see backend documentation)

### Installation
```bash
# Install dependencies
pnpm install

# Create environment file
cp .env.example .env

# Update .env with your backend API URL
# VITE_API_BASE_URL=http://localhost:8000

# Start development server
pnpm run dev
```

### Build for Production
```bash
pnpm run build
```
